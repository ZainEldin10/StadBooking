# ملعبي

واجهة أولية احترافية لإدارة ملعب كرة قدم، تركّز على الحجوزات والفرق والمتجر ولوحة الإدارة.

## ما الموجود الآن

- صفحة عربية RTL متجاوبة بالكامل.
- حجز موعد مع منع الحجز المزدوج داخل الواجهة.
- اختيار مدة الحجز وتسعير متغير حسب الوقت.
- حجوزات قادمة وعدّاد ولاء.
- طلب لاعب وتحديات بين الفرق.
- متجر رياضي مصغر وسلة بسيطة.
- لوحة إدارة بإحصائيات وإجراءات سريعة.

## التطوير المقترح لاحقًا

- تحويل الواجهة إلى Next.js.
- إضافة Backend منظم عبر REST أو tRPC.
- ربط PostgreSQL مع Prisma.
- Authentication آمن للاعبين والإدارة.
- WebSocket أو Realtime لتحديث الأوقات المتاحة مباشرة.
